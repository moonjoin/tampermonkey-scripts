# tampermonkey-scripts

次元饺子个人定制油猴脚本合集，核心是：把网页、B站、Folo、浏览记录里的信息，借助AI大模型能力，自动整理成更好读、可保存、可同步的内容。

- 发布站：https://greasyfork.org/zh-CN/users/1593947-moon-join
- 国内代理：https://home.greasyfork.org.cn/zh-hans/lookup/#?q=moonjoin
- B站主页：https://space.bilibili.com/38389107
- 开源协议：MIT

---

## 现在有 5 个脚本

| 脚本 | 适用场景 | 核心作用 | 是否需要 API |
| :--- | :--- | :--- | :--- |
| B站省流助手 - 字幕AI摘要 Pro | 看 B 站视频 | 自动提取字幕，总结视频和评论，还能判断水视频/软广 | 需要 |
| 饺子 AI 网页摘要助手 | 看文章、新闻、论坛、文档 | 指定网站自动弹出摘要，支持连续追问 | 需要 |
| Folo 网站增强工具 | 用 Folo 读 RSS/文章 | 替换/增强 Folo AI 总结，抓全文、缓存、对话、保存到 flomo | 需要 |
| 网页浏览记录助手 | 记录和复盘浏览行为 | 自动保存浏览记录，推送网页，AI 分析一段时间看了什么 | 可选 |
| 网页一键自动复制 | 日常复制网页文字 | 选中文本自动复制，可附带标题和链接 | 不需要 |

## 1. B站省流助手 - 字幕AI摘要 Pro

文件：`B站省流助手/B站省流助手_-_字幕AI摘要_Pro.user.js`

用途：快速判断一个 B 站视频值不值得看。

关键功能：
- 自动获取 B 站字幕并生成 AI 摘要。
- 支持视频摘要、评论区总结、持续追问。
- 支持多个提示词预设：极简白话、详细笔记、批判分析、行动清单。
- 支持字幕下载、摘要缓存、无字幕时手动上传字幕。
- 支持总结生图、模型切换、自动获取模型列表。
- 支持 flomo 保存和自动加标签。

适合你：
- 不想看完整视频，只想知道结论。
- 想识别广告、水视频、带货内容。
- 想把视频内容变成笔记或行动清单。

## 2. 饺子 AI 网页摘要助手

文件：`AI_网页摘要助手/饺子_AI_网页摘要助手.user.js`

用途：打开指定网页后，自动提取正文并生成摘要。

关键功能：
- 支持指定网站规则，匹配后自动总结。
- 支持通用网页正文提取和 Markdown 渲染。
- 支持连续对话，基于当前网页继续追问。
- 支持多个 API 配置、多模型、多提示词模板。
- 支持按网址绑定不同提示词模板。
- 支持自动复制摘要、发送到 flomo。
- 支持坚果云 WebDAV 同步配置。
- 支持 SPA 路由变化后自动重置和重新总结。

适合你：
- 看长文、论坛、资讯时，只想先看结论。
- 经常用不同模型/提示词处理不同网站。
- 想把网页摘要直接沉淀到 flomo。

## 3. Folo 网站增强工具

文件：`Folo_网站增强/Folo_网站增强工具.user.js`

用途：增强 Folo 网页版阅读体验，用自己的 API 做文章总结。

关键功能：
- 隐藏/替换 Folo 原生 AI 总结入口。
- 支持 Jina Reader、Readability、启发式算法三级抓全文。
- 支持预览模式和全文模式切换。
- 支持自动总结、预加载缓存、摘要缓存。
- 支持后续对话、复制完整对话。
- 支持保存到 flomo。
- 支持多 API 配置和坚果云 WebDAV 同步。
- 支持解除文章复制限制，提供 Copy 按钮。

适合你：
- 用 Folo 读 RSS，但想用自己的模型总结。
- 想抓取原文全文，而不是只总结 Folo 预览。
- 想把 RSS 阅读内容整理到 flomo。

## 4. 网页浏览记录助手

文件：`网页浏览记录自动推送/网页浏览记录自动推送.user.js`

用途：自动记录你看过什么网页，并支持推送和 AI 复盘。

关键功能：
- 自动保存网页标题、链接、访问时间。
- 支持停留时间阈值、冷却时间、排除域名。
- 支持当前页自动推送或手动推送。
- 支持 Telegram Bot 推送。
- 支持飞书 Webhook 推送。
- 支持 AI 分析浏览记录：未读、今天、本周、本月、全部、自定义时间段。
- 内置分析模板：时段总结、关注点分析、高频内容提取、自由提问。
- 支持浏览记录导出、删除、清理。
- 支持坚果云增量同步，并和网页摘要助手、Folo 增强工具共享部分配置。

适合你：
- 想知道自己最近到底看了什么。
- 想把重要网页自动发到 Telegram 或飞书。
- 想让 AI 根据浏览记录总结兴趣、重点和待处理内容。

## 5. 网页一键自动复制

文件：`网页选中文本自动复制/选中文本自动复制.user.js`

用途：选中网页文字后自动复制，减少复制步骤。

关键功能：
- 选中文本后自动复制到剪贴板。
- 可开启“附带出处信息”，复制内容后追加网页标题和 URL。
- 支持 `Alt + X` 快捷键开关。
- 悬浮按钮可拖拽移动。
- 有复制成功提示和按钮状态反馈。
- 自动跳过输入框、文本框、下拉框等可编辑区域。

适合你：
- 经常摘录网页文字。
- 复制时希望自动带上来源链接。
- 不想每次选中文本后再按 `Ctrl + C`。

## 使用前准备

1. 浏览器安装 Tampermonkey / 篡改猴（手机电脑全平台可装，iOS可用Stay）。
2. 从 GreasyFork 或本仓库安装对应 `.user.js`。
3. AI 类脚本需要填写兼容 OpenAI 格式的接口：
   - API 地址，一般类似 `https://xxx/v1/chat/completions`
   - API Key
   - 模型名
4. 需要保存笔记时，填写 flomo API。
5. 需要跨设备同步时，填写坚果云 WebDAV 账号和应用密码。

## 隐私提醒

- API Key、flomo 地址、坚果云账号等配置保存在浏览器脚本存储里。
- AI 总结会把网页内容、字幕、评论或浏览记录发送到你配置的 API 服务。
- 浏览记录助手会记录访问历史；如果开启 Telegram、飞书或坚果云，会把相关数据发送到对应服务。
- 不想上传的内容，请关闭自动总结、自动推送或云同步。

## 推荐模型

总结类任务优先选 Flash / 轻量模型：速度快、成本低，适合摘要、提炼、改写。

当前默认配置偏向：
- `mimo-v2-flash`
- `deepseek-v4-flash`
- `gemini-3-flash-preview`
- `claude-haiku` 类模型

原则：先用快模型做摘要；只有需要深度分析、复杂推理时再换强模型。
